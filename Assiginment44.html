Logical and Comparison Operators in JavaScript

Logical AND (&&)

Returns true if both operands are true.

let a = true;
let b = false;

console.log(a && b); // false
console.log(a && !b); // true
console.log(a && (3 > 1)); // true
Logical OR (||)

Returns true if at least one operand is true.

let a = true;
let b = false;

console.log(a || b); // true
console.log(b || (3 > 1)); // true
console.log(b || (1 > 3)); // false
Logical NOT (!)

Returns true if the operand is false, and vice versa.

let a = true;
let b = false;

console.log(!a); // false
console.log(!b); // true
console.log(!(3 > 1)); // false

Equal (==)

Checks if the values of two operands are equal. Performs type coercion if necessary.

console.log(5 == '5'); // true
console.log(5 == 5); // true
console.log(5 == 6); // false
Strict Equal (===)

Checks if the values and types of two operands are equal. No type coercion is performed.

console.log(5 === '5'); // false
console.log(5 === 5); // true
console.log(5 === 6); // false
Not Equal (!=)

Checks if the values of two operands are not equal. Performs type coercion if necessary.

console.log(5 != '5'); // false
console.log(5 != 5); // false
console.log(5 != 6); // true
Strict Not Equal (!==)

Checks if the values and types of two operands are not equal. No type coercion is performed.

console.log(5 !== '5'); // true
console.log(5 !== 5); // false
console.log(5 !== 6); // true
Greater Than (>)

Checks if the left operand is greater than the right operand.

console.log(5 > 3); // true
console.log(5 > 5); // false
console.log(3 > 5); // false
Greater Than or Equal (>=)

Checks if the left operand is greater than or equal to the right operand.

console.log(5 >= 3); // true
console.log(5 >= 5); // true
console.log(3 >= 5); // false
Less Than (<)

Checks if the left operand is less than the right operand.

console.log(5 < 3); // false
console.log(5 < 5); // false
console.log(3 < 5); // true
Less Than or Equal (<=)

Checks if the left operand is less than or equal to the right operand.

console.log(5 <= 3); // false
console.log(5 <= 5); // true
console.log(3 <= 5); // true
