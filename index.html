ECMAScript 5.1 (ES5.1) - June 2011
*Minor revision to align with ISO/IEC standards.

*Also known as ECMAScript 6 (ES6) or ECMAScript Harmony.
*Major update introducing many new features:
*Block-scoped variables with let and const.
*Arrow functions for concise function syntax.
*Template literals for string interpolation.
*Destructuring assignment for unpacking values from arrays or properties from objects.
*Default parameters in functions.
*Rest and spread operators for arrays and function arguments.
*Modules with import and export syntax.
*Promises for asynchronous programming.
*Symbol data type for creating unique identifiers.
*Iterators and generators for custom iteration protocols.
*Map, Set, WeakMap, and WeakSet collections.

Post-ES6 Developments
Following ES6, ECMAScript has moved to a yearly release cycle with incremental improvements and new features added each year.
